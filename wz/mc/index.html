方块查询功能
用于查询方块破坏记录,箱子拿取记录

输入/co i 后点击方块或者箱子即可进行查询

再次输入/co i 后即可关闭查询系统

[ 注意,指令是 /co空格i 而不是/coi ]

——————————————————————————————————————————

物品上锁教程
对于容器类只需放置木牌即可自动上锁

对于门类放置木牌在门上或周围的方块上自动上锁

其他物品你可以自己添加锁定信息 按照生成木牌的格式填写即可

那如何信任其他玩家给其权限呢

在第二步的 <文本> 内容中添加你要信任的玩家名即可

也可以再放置一个木牌自动生成 [所有人] 标签 任何人都可以访问该容器

注意:请在箱子下方也放上容器并锁起来,避免他人使用漏斗偷东西!!

——————————————————————————————————————————

领地创建教程
自己的建筑物一定要圈地保护起来,这很重要

制作一根木锄 找到一块区域对角的两个方块

左键第一个方块 右键第二个方块

输入/res create [ID] 一个领地就创建完成了

若实在无法理解如何创建领地群文件中也有视频教程供参考

其他相关帮助:

/res remove 名字 删除领地

/res set 设置领地权限

普通玩家领地数量上限: 3

普通玩家领地大小上限: 150*150

普通玩家领地创建价格: 0.1$/方块(游戏币)

更多指令帮助查看请在游戏中输入/res ?

——————————————————————————————————————————

商店创建教程
首先放置一个箱子 然后把你想出售的物品左键一下箱子

最后直接在聊天栏中输入你想出售的价格即可创建一个商店

相关帮助:

/qs buy 切换为收购模式

/qs sell 切换为出售模式

/qs price 更改新的价格

商店数量上限: 无限

输入/qs 查看更多帮助

🎮 Minecraft 坞中客 - 指令列表

游戏版本：1.21.8
开服时间：2025-10-13

📜 服务器公告

坞中客 - DvidNanx
2025/10/26 18:26

  • 📅 坞中客 1.21.8 存档于 2025 年 10 月开始
  • 🤝 承诺永不删档（无法预测的风险除外）

⚠️ 重要规则

  • 🗑️ 自动删除 35 天内未登录玩家的全部领地
  • 🔄 采用刷新区块替代换周目
    • 每个假期前重置 全部未圈地区块
    • 刷新 全服生物数据（会提前通知）
  • 🧹 区块自动清理机制
    • 若区块 未圈地且长期无人加载 → 自动删除（不会通知）

🔧 常用指令

  • /a —— 打开主菜单
    • 生存区菜单：大写 /A 或小写 /a 均可直接打开
    • 生电区菜单：必须使用小写 /a
  • <s>打开服务器菜单(生电区/ad)</s> —— ❌ 已失效
  • /spawn —— 回到主城

💍 结婚系统 (/marry)

—-< Marriage Command Help >—-

  • /marry gift —— 送物品给你伴侣
  • /marry <player> —— 请求与另一位玩家结婚
  • /marry pvp on/off —— 开启或关闭PVP
  • /marry list [page] —— 查看所有已婚人士的菜单
  • /marry seen —— 检查你伴侣最后登入的时间
  • /marry home —— 传送到你们的家
  • /marry chat —— 开启伴侣聊天
  • /marry heal —— 把你的生命给了你的伴侣
  • /marry sethome —— 设置你们的家
  • /marry divorce —— 和你现在的伴侣离婚
  • /marry tp —— 传送到你的伴侣

💰 转账系统（XConomy）

  • /pay [ID] 数量 —— 给玩家转账

💳 相关指令：

  • balance / money —— 查询余额
  • balance/money <玩家> —— 查询 <玩家> 余额
  • pay <玩家> <金额> —— 转账给 <玩家> <金额>
  • balancetop —— 查询余额排行榜
  • paytoggle —— 切换是否接受转账

🏠 家系统

  • /sethome [ID] —— 设置一个家
  • /home [ID] —— 回到这个家
  • /delhome [ID] —— 删除这个家

✅ 普通玩家最多可设置 3 个家
🌍 家可以设置在不同维度：主世界 / 地狱 / 末地（包括一区、二区、生电区）
⏱️ 家传送无冷却CD

🔄 传送请求

  • /tpa [ID] —— 申请传送至该玩家（你 → 他）
  • /tpahere [ID] —— 让玩家传送到你这里（他 → 你）
  • /tpaccept —— 接受玩家传送请求

👤 玩家交互

  • /skin [ID] —— 切换他人皮肤（离线可用）

📦 物品与背包查看：

  • [i] —— 展示手中物品
    • 工具/武器：显示附魔属性
    • 潜影壳：显示内部物品
    • 地图画：可预览
    • 书与笔：查看文字内容
  • [inv] —— 展示背包物品
    • 显示人物头像、经验值、装备、背包物品
    • 潜影壳可单独打开
  • [ender] —— 展示末影箱（功能同背包）

💬 聊天频道切换：

  • /g —— 切换全服聊天频道
  • /l —— 切换本地聊天频道

🔄 一二区互通 | 生电区单独

🔍 方块查询功能

  • 输入 /co i 后点击方块或箱子即可查询
  • 再次输入 /co i 关闭查询系统

📌 注意：指令是 /co 空格 i，不是 /coi
可查询：

  • 箱子存放信息
  • 方块放置与破坏记录

🏗️ 领地创建教程（圈地帮助）

  1. 制作一根 木锄
  2. 找到区域对角的两个方块
    • 左键第一个方块，右键第二个方块
  3. 输入 /res create [ID] —— 创建完成！

📹 若无法理解，群文件中有视频教程参考

📏 领地限制（普通玩家）：

  • 数量上限：3 个
  • 大小上限：150×150
  • 创建价格：0.1$/方块（游戏币）
  • 会员折扣：0.08$ / 0.07$ / 0.05$（按等级）

📋 领地总指令

❓ 帮助

  • /res ? —— 查看帮助

🏗️ 创建与删除

  • /res create [领地名] —— 创建领地
  • /res remove <领地名> —— 删除领地
  • /res subzone [子领地名] —— 创建子领地（上限：3）
  • /res auto [领地名] [范围] —— 自动创建最大领地
  • /res select [x] [y] [z] —— 选取保护区域（Z=高度）
  • /res select size —— 显示选区大小
  • /res select cost —— 显示选区价格
  • /res select auto [玩家] —— 自动选择区域 （预览扩展）
  • /res expand [数值] —— 向面对方向扩展（实际生效）
  • /res contract <领地> [缩小单位] —— 向面对方向缩小
  • /res select shift [数值] —— 选区平移
  • /res select vert —— 高度从天空到基岩
  • /res select sky —— 高度到天框
  • /res select bedrock —— 高度到基岩
  • /res select chunk —— 选择当前区块
  • /res select worldedit —— 使用 WorldEdit 选区

🧩 区域管理

  • /res area add [领地] [区域ID] —— 添加物理区域

  • /res area remove [领地] [区域ID] —— 移除物理区域

  • /res area replace [领地] [区域ID] —— 替换物理区域

    ✅ 可与其他领地区域重叠

  • /res setmain <领地> —— 设置主领地

🔐 权限与名称

  • /res padd <领地名> [玩家] —— 添加基本权限
  • /res pdel <领地名> [玩家] —— 删除基本权限
  • /res pset <领地名> [玩家] [权限] [true/false/remove] —— 设置玩家权限
  • /res set <领地名> [权限] [true/false/remove] —— 设置领地全局权限
  • /res pset <领地> [玩家] removeall —— 删除玩家所有权限
  • /res rename [旧名称] [新名称] —— 重命名领地

    📌 子领地需用全名：主领地.子领地

  • /res mirror [原领地名] [目标领地名] —— 复制权限（需为两领地所有者）
  • /res gset <领地名> [组名] [标志] [true/false/remove] —— 设置组权限
  • /res reset <领地> —— 重置所有权限
  • /res lset <领地> [blacklist/ignorelist] [材料] —— 加入黑名单（禁止放置）
  • /res lset <领地> Info —— 忽略名单选项

ℹ️ 领地信息

  • /res message <领地名> [enter/leave] [信息] —— 自定义进出消息
  • /res info —— 显示当前领地信息（若在外，显示自己领地）
  • /res list [玩家] —— 列出玩家拥有的领地
  • /res listall <页码> —— 列出所有领地
  • /res area list [领地] <页码> —— 列出物理区域
  • /res area listall [领地] <页码> —— 列出所有区域坐标详情
  • /res lists —— 预定义权限列表
    • add / remove / apply / set / pset / view —— 管理列表
  • /res listhidden <玩家> <页码> —— 列出隐藏领地

🚀 传送相关

  • /res tp [领地名] —— 传送到指定领地
  • /res tpset —— 设置领地传送点
  • /res unstuck —— 移动到领地外
  • /res show <领地> —— 显示领地边界
  • /res give [领地名] [玩家] —— 赠送领地（玩家须在线，且你是所有者）

🔒 物品上锁教程

  • 容器类：放置木牌即可自动上锁
  • 门类：在门上或周围方块放木牌自动上锁
  • 其他物品：按木牌格式填写锁定信息

👥 添加信任玩家：

  • 在第二行 <文本> 中添加你要信任的玩家名
  • 放置新木牌自动生成 [所有人] 标签（公开访问）

⚠️ 重要提醒：请在箱子下方也放上容器并锁起来，避免他人使用漏斗偷东西！

🛒 商店创建教程（QuickShop）

  1. 放置一个 箱子
  2. 左键箱子放入想出售的物品
  3. 聊天栏输入价格 → 商店创建成功！

📦 商店类型：

  • 箱子商店：支持木桶或箱子
  • 模式切换：
    • /qs buy —— 切换为收购模式
    • /qs sell —— 切换为出售模式
  • 其他指令：
    • /qs price —— 更改价格
    • /qs find —— 查找附近指定物品商店
    • /qs remove —— 删除看向的商店
    • /qs item —— 更改商店物品

📌 左键牌子：输入数量（all 全买），可预览物品
📌 右键牌子：打开控制面板（切换模式）
💰 商店数量上限：无限
